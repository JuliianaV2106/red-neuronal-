## Descripción
Este proyecto implementa una red neuronal básica desde cero utilizando
regresión logística, con el objetivo de comprender el funcionamiento interno
del aprendizaje automático, incluyendo la propagación hacia adelante,
la propagación hacia atrás y el descenso de gradiente.

La implementación se basa en el notebook proporcionado por el docente
(`0_LR_NN_V2.ipynb`) y se adapta a un dataset diferente, cumpliendo con los
requisitos del ejercicio académico.

## Contenido del Repositorio

- `notebooks/Red.ipynb`: implementación principal del ejercicio.
- `notebooks/0_LR_NN_V2.ipynb`: notebook base de referencia.
- `data/train.csv`: dataset utilizado.
- `docs/RED_NEURONAL.pdf`: explicación teórica del algoritmo.
